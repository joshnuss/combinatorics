# Combinatorics

Utility functions for generating combinatoric.

### Supported operations

- binary combinations
- product
- powerset
- n combinations
- permutations

[Documentation](https://hexdocs.pm/combinatorics)

## Installation

Add `combinatorics` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:combinatorics, "~> 0.1.0"}
  ]
end
```

Run `mix do deps.get, deps.compile`

Have fun!
